World Wind v2.0.0 Https Patch
January 2017

The updated layer configuration files in this package allow the World Wind 2.0.0 client to utilize
the new https endpoints.

To use this patch, place the contents in your projects classpath and set the following property:

-Dgov.nasa.worldwind.app.config.document=worldwind-2.0.0-https-patch/worldwind-https-patch.xml

Further information can be found at:
https://goworldwind.org/developers-guide/common-problems/
